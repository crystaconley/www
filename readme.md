### Setup

- make sure [node.js](http://nodejs.org) and [roots](http://roots.cx) are installed
- clone this repo down and `cd` into the folder
- run `npm install`
- run `roots watch`

### Deploying

- If you just want to compile the production build, run `roots compile -e production` and it will build to the public directory.
- To deploy this site, just make changes on master, and push it up. Aerobatic will take care of the rest.
